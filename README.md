# 🚀 DigiTools Platform

## 📌 Overview

**DigiTools Platform** is a modern web application where users can explore and purchase premium digital tools designed to enhance **productivity** and **creativity**. From resume builders to AI writing tools and development utilities, the platform offers a curated collection of powerful resources for professionals and learners alike.

---

## ✨ Key Features

### 🛒 1. Smart Product Marketplace

Browse a wide range of digital tools like:

* Resume builders
* UI design kits
* AI content generators
* Developer productivity tools
  Each product includes detailed descriptions, pricing, and feature highlights.

---

### ⚡ 2. Interactive Cart System

* Add/remove tools easily
* Real-time cart updates
* Dynamic total price calculation
* Clean and responsive UI for better user experience

---

### 🔔 3. User Feedback with Toast Notifications

* Instant notifications using React-Toastify
* Alerts for adding/removing items
* Improves usability and interaction feedback

---

## 🛠️ Technologies Used

* **React.js** – Component-based UI development
* **Tailwind CSS** – Utility-first styling
* **DaisyUI** – Pre-built UI components for faster design
* **JavaScript (ES6+)** – Core logic and functionality
* **React-Toastify** – User notifications
* **JSON** – Product data management

---


## 🎯 Purpose

The goal of DigiTools Platform is to provide a **centralized hub** for high-quality digital tools that help users:

* Work smarter
* Save time
* Boost creativity
